# Student Record Management System (Python)

This is a beginner-friendly Python project built using:
- Lists
- Dictionaries
- File handling (JSON)

## Features
- Add student details
- View all students
- Search student by roll number
- Data stored permanently using JSON file

## How to Run
1. Download the Python file
2. Run using: python student_record.py

This project was built as part of learning Python fundamentals.
